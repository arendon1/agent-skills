# Deep Research Scripts

TypeScript utilities for automated research.

## Scripts

- **academic-researcher.ts**: Find peer-reviewed papers via Google Scholar
- **company-analyzer.ts**: Deep dive company analysis
- **web-researcher.ts**: Multi-source web research
- **report-generator.ts**: Generate structured reports

## Running Scripts

All scripts use Bun runtime:

```bash
bun scripts/[script-name].ts [args]
```

See parent SKILL.md for detailed usage examples.

## Development

To test the scripts:

```bash
bun test deep-research.test.ts
```
