# Publishing Strategy Guide

## Journal Selection

### Quality Indicators

* Reputation in field.
* Indexing (Web of Science, Scopus).
* Acceptance rate/Review time.

### Red Flags (Predatory Journals)

* Rapid "review" (days).
* Aggressive solicitation.
* "Pay to publish" without clear OA model.

## Cover Letter Template

```markdown
Dear Dr. [Editor],

We are pleased to submit "[Title]" for consideration in [Journal].

**Summary**: [What you found].
**Significance**: [Why it matters].
**Fit**: [Why this journal].

**Declarations**: Original work, all authors approved.

Sincerely,
[Author]
```

## Responding to Peer Review

### Strategy

* **Must Address**: Methods, clarity, missing lit.
* **Negotiate**: Restructuring, additional analysis.
* **Pushback**: Out of scope requests (polite).

### Response Template

* **Comment**: [Reviewer's point]
* **Response**: We agree/disagree because...
* **Change**: "New text added..." (p. X, line Y).
