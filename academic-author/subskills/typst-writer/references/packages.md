# Package Usage

When needing specialized functionality:
1. Search for packages at https://typst.app/universe/
2. Import with `#import "@preview/package:version"`
3. Consult package documentation for API

## Popular Examples

| Function | Package |
| :--- | :--- |
| **Drafting** | `drafting` (annotations), `gentle-clues` (admonitions) |
| **Tables** | `tablem` (markdown-like), `tablex` (advanced) |
| **Diagrams** | `cetz` (drawing), `fletcher` (graphs/flowcharts) |
| **Charts** | `timeliney` (Gantt), `herodot` (timelines), `lilaq` (plots) |
| **Formatting** | `showybox` (boxes), `itemize` (lists), `zebraw` (code) |
| **Academic** | `lovelace` (algorithms), `eqalc` (math) |
| **Slides** | `polylux` |

## Finding Packages
- **Typst Universe**: https://typst.app/universe
- **GitHub Search**: `gh search repos --language "typst" topic`
