# Prompt Engineering Tips

**Effective Prompts for Scientific Diagrams:**

## 1. Be Specific About Layout
*   ✅ "Flowchart with vertical flow, top to bottom"
*   ✅ "Architecture diagram with encoder on left, decoder on right"

## 2. Include Quantitative Details
*   ✅ "Neural network with input layer (784 nodes), hidden layer (128 nodes)"
*   ✅ "CONSORT flow: n=500 screened, n=350 randomized"

## 3. Specify Visual Style
*   ✅ "Minimalist block diagram"
*   ✅ "Technical schematic with engineering notation"

## 4. Request Specific Labels
*   ✅ "Label arrows with activation/inhibition"
*   ✅ "Show time progression with timestamps"

## 5. Color Guidelines
*   ✅ "Use colorblind-friendly colors"
*   ✅ "Color-code by function: blue for input, green for processing"

## Global Guidelines
*   Avoid vague prompts like "Make a flowchart".
*   Always specify the **Type** (Flowchart, Circuit, Pathway).
*   Always list **Components** and **Connections**.
