# Best Practices

## Design Principles
1.  **Clarity over Complexity**: Remove unnecessary elements.
2.  **Consistent Styling**: Use standard fonts (Sans-serif, 7-8pt) and line weights.
3.  **Accessibility**: Use Okabe-Ito palette. Ensure grayscale readability.
4.  **Vector Format**: Prefer PDF/SVG for publication.

## Integration
*   **LaTeX**: Use `\includegraphics{}`.
*   **Caption**: Describe all elements.
*   **Consistency**: Match style across all figures in manuscript.

## Quality Checklist
*   [ ] No overlapping elements.
*   [ ] Text legible (min 7pt).
*   [ ] Colorblind safe.
*   [ ] High resolution (300+ DPI).
